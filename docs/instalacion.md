# Guía de Ejemplo

Bienvenido a la documentación creada con **MkDocs**.

## Introducción

MkDocs permite crear sitios de documentación estáticos utilizando archivos Markdown.

### Lista de características

- Fácil de configurar.
- Compatible con temas personalizados.
- Generación automática de navegación.

## Ejemplo de código Python

```python
def saludar(nombre):
    print(f"Hola, {nombre}")

saludar("Mundo")
```
